## 第一课：《一个人，就是一家公司，然后呢？》——人物杂志

**拆解前先问自己**：如果让AI写这个选题，会怎么写？

AI会写：
> 随着AI技术的普及，"一人公司"正在成为新趋势。AI让个人的力量成倍放大，一个人就能完成一个部门的工作。但这条路并不容易……

**而原文是怎么写的？**

四个人的故事，每个都按一条完整的线走：

### 火哥（43岁连续创业者）的叙事线

**【起点：一个具体的动作】**
> 前几天我处于极度亢奋，我的新公司只有我一个人类，但我有8只AI龙虾。

不对——**真正的起点在这里**：
> 一切都起因于半个月前，我本来只是想拿AI做一个实验。我以前是旅游背景的，试着训练了一只"龙虾"去帮我们做美国签证的面签报告。

**注意：他不是从"大背景"开始的。他是从"半个月前，我想做一件事"开始的。**

**【转折1：踩坑】**
> 我充了1000美金，预期够跑一个月。结果三天就烧没了。
> ——现实世界给的第一巴掌

**【转折2：又踩坑】**
> 每天想到好点子，第二天就能用AI跑出原型。但现实世界的备案流程死死地等在那儿，快不起来。

**【金句收束】**
> 以前创业是老板嫌员工干活慢，现在倒好，AI把活儿干完了，我这个唯一的人类反而成了全公司"拖后腿"的那个。

**【结尾的观点——不是总结，是个人感受】**
> 在这个充满不确定的转型期里，虽然也会被账单吓到，会被备案卡住，但至少，我看到了一个全新的世界。

**→ 这条线的结构：一个具体动作 → 遇到问题 → 再遇到问题 → 一句点题 → 个人收尾**

### 为什么这样写读起来"有人味"？

1. **每一个情绪都有原因** — 他不是说"我很焦虑"，他是说"我看着AI花了90分钟写出一份高盛风格的投研报告"然后才"护城河确实在消失"

2. **信息是"附在故事上"的** — 不是先讲概念"一人公司怎么怎么样"，是在火哥的故事里自然而然带出来的

3. **没有人物的"总结陈词"** — 没有"这告诉我们一人公司虽然苦但有希望"，他只是说"我看到了一个全新的世界"，判断留给读者

4. **每一段都是"这个人的话"** — 不是记者转述的口气，是这个人自己在说

## 第二课：我之前的文章vs真人文章，逻辑对比

### 真人逻辑（以火哥为例）

```
我做了一个具体的事（训练龙虾做签证）
→ 这件事有了结果（能处理50本/天）
→ 又遇到了新问题（1000美金三天烧完）
→ 又突破（一天跑出原型）
→ 又遇到新问题（备案卡住）
→ 点题（我成了拖后腿的）
→ 收尾的感受（但我看到了新世界）
```

**这是一条"经历→感受"的线。** 读者的感觉是：我在听一个人跟我聊天。

### AI逻辑（我的前几版）

```
诺奖得主说AI会取代人类工作（观点）
→ 柳州一个朋友转行做机器人训练师（例子1）
→ 詹姆斯·黑尔跨界做AI医疗（例子2）
→ 马达加斯加女孩做数据标注（例子3）
→ 所以我们要动起来（结论）
```

**这是一条"观点→论据A→论据B→论据C→结论"的线。** 读者的感觉是：你在证明给我看。

### 关键区别

| 真人写法 | AI写法 |
|---------|--------|
| 从一件事出发 | 从一个观点出发 |
| 素材是"我想到的" | 素材是"我找来的" |
| 结论是自然流露的 | 结论是事先设定好的 |
| 读者听完想"嗯……" | 读者听完想"哦" |
| 有犹豫、有矛盾、有不完美 | 每一步都"对"，没有起伏 |
| 结尾留给读者去想 | 结尾告诉你"该怎么办" |

## 第三课：我到底该怎么写才对？

**核心原则：不再"先搭框架再填素材"，而是"先找到那条经历线"。**

### 写一篇文章的正确步骤（应该是）

**第1步：找到你的起点——一件具体的事**
- ❌ "最近我在想一个人能不能活成一家公司"（抽象）
- ✅ "那天一个做自媒体的朋友跟我吃饭，他说他一个人干了五个人的活"（具体）

**第2步：顺着这件事，往下自然发展**
- 这件事让我有什么感受？
- 这个感受让我想到了什么？
- 想到了什么别的事/别人？

**第3步：写到哪算哪，不用预设结论**
- 写到最后，你自然知道你想说什么
- 如果你不知道，说明你还没想清楚

**第4步：回头再调结构和节奏**
- 不是先搭结构再填内容——是内容写出来了，自然就有结构了

### 实战：如果我重新写诺奖得主那篇文章

**真正的起点应该是：**
> 我最近在刷招聘软件，越刷越没底。

而不是：
> 诺奖得主说AI会取代人类的工作。

**为什么？**
因为"刷招聘软件越刷越没底"是**一件具体的事**，是我自己经历的。而"诺奖得主说"是**一个别人的观点**，跟我没有直接关系。

**从具体的事出发，自然就会想到：**
- 刷到岗位要求越来越高 → 心里慌 → 刷到诺奖得主的采访 → 他说的话让我想了很久 → 想到了朋友的事 → 想起了马达加斯加那个女孩 → 最后我自己想通了什么

这条线不是"拼"的——它是一条自然流淌的线。

---

## 总结：我这次学到的三件事

**1. 不要从观点出发，从一件具体的事出发。**
"最近我在找工作"比"诺奖得主说"好一万倍。

**2. 素材不是"例子"，是"我想到了什么"。**
"想到了一个朋友"比"举个例子"好一万倍。

**3. 结尾不是总结，是我自己的收束。**
"你呢？"比"所以我们应该动起来"好一万倍。
