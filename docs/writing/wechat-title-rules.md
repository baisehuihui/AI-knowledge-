## 爆文标题五大规律

### 1. 数字要具体，不要模糊

| ❌ 模糊 | ✅ 具体 |
|---------|---------|
| 很多人 | 20万人、4800万人 |
| 一些公司 | 3人团队 |
| 大幅下降 | 骤降50% |
| 成本变低 | 3000元成本、600元/分钟 |

**原理**：具体数字触发大脑的"真实感"判断。读者看到"20万人"比"很多人"更愿意点开。

### 2. 制造冲突/反转

爆文标题的核心公式之一：**你以为A → 实际上是B**

案例：
- "淘汰你的不是AI，是比你更会提问的同事"（你以为：AI淘汰你 → 实际：同事淘汰你）
- "3年前被骂作弊，2026年让4800万人真香"（你以为：AI不行 → 实际：人人都在用）
- "同一个行业，他在种地，他日入6万"（你以为：全行业完蛋 → 实际：有人赚更多）

**原理**：人天生对"意料之外"的信息敏感，这是注意力机制的底层逻辑。

### 3. 给出明确价值承诺

读者点开一篇文章前会问：**"这对我有什么用？"**

标题直接回答这个问题：
- "普通人的翻身机会终于来了"
- "普通人如何避免出局"
- "普通人的最后一个机会"

⚠️ 注意：价值承诺必须可兑现，否则点了就关，还会取关。

### 4. 时间锚点制造紧迫感

- "2026年最残酷的职场真相"（锚定到现在，暗示紧急）
- "3年前...2026年..."（对比时间差，暗示变化很大）
- "春节刚过"（锚定到最近事件，暗示刚发生）

**原理**：人更容易对"当前"和"即将发生"的事情做出反应。

### 5. 找"小众但高痛点"的角度

写烂的角度：
- "AI来了"（太泛）
- "AI取代工作"（被写过一万次）

有人另辟蹊径的角度：
- "一个人就是一家公司"（不说AI，说人的能力变了）
- "别卷文笔了，爆款文章是产品思维"（不说AI写作，说文笔不重要了）

**关键**：同一个话题，找一个没人写过的新切面。

## 标题自测清单

写完标题后，问自己五个问题：

1. **有具体数字吗？** → 没有就加
2. **有反转/冲突吗？** → 没有就找反差
3. **读者知道"这对他有什么用"吗？** → 不知道就改
4. **时间锚点够近吗？** → 不够就往前拉
5. **这个角度别人写过吗？** → 写过就换切面

## 📌 核心定位：针对目标读者的标题心法

### 目标读者画像
25-35岁职场人，有AI兴趣但不知道怎么变成生产力，有职场焦虑

### 标题方向
- **B方向：拉近距离，讲"跟我差不多的人的故事"**
- 核心公式：（跟我差不多的人）+（遇到跟我一样的问题）+（发生了什么/做了什么选择）
- 禁止：远方的传奇故事、日入几万的神话、跟普通人无关的大厂新闻

### 读者真正关心的三件事
1. AI跟我有什么关系？
2. 跟我一样的人是怎么做的？
3. 我现在做还来得及吗？

### 标题写作原则

#### 核心原则：读者点的不是标题，是自己的欲望

读者看到的不是文字，是这些问题：
- "这关我什么事？" → 标题要直接回答
- "这说的是我吗？" → 身份标签要对
- "看完对我有用吗？" → 结局暗示要好

#### 7个人性触发点（适合咱们定位的）

| 人性 | 原理 | 适合咱们的用法 |
|------|------|----------------|
| 自私心理 | "关我什么事" | 利益承诺+身份标签：你做XX的/XX岁/干了XX年 |
| 共鸣感 | "原来你也这样" | 身份标签+情绪关键词：做了XX年，但…… |
| 恐惧心理 | "再不行动就晚了" | 危机预警+解决方案：不会AI的人正在被…… |
| 好奇心理 | "然后呢？" | 反常识+场景化：同事用AI一天干完我三天的活 |
| 窥探欲 | "别人不知道的事" | 悬念+信息差：公司调整名单里，最先…… |
| 捡漏心理 | "不能错过" | 限时感+超预期价值：最后的机会/现在还来得及 |
| 优越感 | "我比别人懂" | 稀缺结论+实用价值：3个方法/一个技巧/一件事 |

### 禁忌清单

❌ **不要**：远方的传奇（"演员种地""日入6万" → 太远）
❌ **不要**：大厂新闻（"OpenAI又融资了" → 关读者啥事）
❌ **不要**：贩卖焦虑不给答案（"你的岗位会被AI替代" → 然后呢？）
❌ **不要**：技术术语堆砌（"GPT-5多模态能力突破" → 看不懂）
❌ **不要**：模糊用词（"很多人""不少""据说" → 不可信）

✅ **要写**：普通人的真实选择（"25岁的同事，被AI替代后干了什么"）
✅ **要写**：可验证的信息（"据XX报道""数据显示" → 建立信任）
✅ **要写**：看完有收获（具体方法、可执行的建议）

## 优秀案例拆解

### 案例1："AI替代真相：真正危险的不是人，而是'工具人'"

**好在哪里**：
- 反转结构："不是A，是B" —— 先否定一个常识，给出新结论
- 身份标签："工具人" —— 精准击中职场人的自我认知
- 冲突制造：AI vs 人 → 人与自己的思维惰性

### 案例2："淘汰你的不是AI，是比你更会提问的同事"

**好在哪里**：
- 反常识：把威胁从"机器"拉回到"人"
- 身份暗示：读者自动带入"我是不是那个不会提问的人？"
- 制造焦虑后有出口：暗示"学会提问就能赢"

### B方向参考标题（基于咱们定位）

按核心公式写的示例：
- "做了5年设计，突然发现同事用AI半天干完我两天的活"
- "28岁，月薪从1.5万降到8千：一个普通人的AI生存实录"
- "我身边那个最先学AI的同事，半年后怎么样了"
- "2026年了，还在手动做报表的人，老板已经不想等了"
- "3个月前我还在焦虑被AI替代，现在……"

## 本文素材可取的标题方向

基于"同一个行业，他在种地，他日入6万"的核心素材：

**反转型**：同一个行业，他在种地，他日入6万——AI清掉的不是人，是中产
**价值型**：3个真实故事告诉你：2026年你的工作还在不在AI的射程内
**时间锚定型**：2026年春节后，短剧圈发生了两件相反的事
**冲突型**：降薪被拒、回家种地、日入6万——同一波AI浪潮下的三种结局
