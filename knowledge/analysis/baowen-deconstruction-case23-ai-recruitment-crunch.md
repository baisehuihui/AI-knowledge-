# 爆文拆解案例23：AI冲击招聘市场，打工人更卷了？

**来源**：Tech星球（网易号） - 2026年4月29日  
**作者**：张宁洢  
**类型**：行业调查/社会观察  
**精拆时间**：2026-05-05（第3天·观点表达维度预习）

---

## 一、文章结构

### 三问式结构

```
小标题1：不会养虾的直接淘汰——AI成面试硬通货
→ AI使用能力直接影响面试通过率和薪资水平
小标题2：AI改变招聘选人标准
→ 创意行业从"作品集权重最高"变为"销售能力≈专业能力"
小标题3：前端员工缩减七倍——AI能代替人工吗？
→ 初级岗位减少，AI+岗位暴增
```

**结构特点**：三个问题按"程度递进"排列——从"要不要会AI"→"怎么用AI"→"AI会不会取代我"。问号句式一是制造悬念，二是引导读者对号入座。

---

## 二、素材组织手法

### 手法1：多源采访+群像拼接

不同于一篇单一视角的文章，这篇采用"猎头说+HR说+程序员说+广告从业者说+求职者说"的**五方视角**：
- 猎头——看市场整体趋势
- HR——看招聘标准怎么变
- 从业者——看岗位实际怎么变
- 求职者——看面试怎么变
- 校招HR——看应届生怎么应对

**可学**：覆盖"供需两端"——招聘方（HR/猎头）+ 求职方（员工/应届生），让读者无论站在哪一边都能找到自己。

### 手法2：对比嵌套

```
三组对比：
├── 效率对比：以前7-8个前端 → 现在1个前端
├── 薪资对比：不要求AI能力15-40k → 要求AI能力25-50k → 大模型PM 40-75k
├── 要求对比：以前看作品集 → 现在看销售能力+资源
```

对比不是平铺直叙，而是层层递进——效率→薪资→要求，从"客观事实"到"个人利益"，逐步靠近读者。

### 手法3：金句点缀

- "AI全民化带来便捷的同时，也带来了焦虑"（主标题下第一句，定调）
- "学会与AI合作成了打工人新的'生存之道'"（点题）
- "人被夹在两个算法之间，努力证明自己'比AI更像人'"（情感共鸣）

3个关键金句分别放在开头、中段、结尾附近。

## 三、观点表达手法

| 手法 | 原文示例 | 分析 |
|------|---------|------|
| **直接提问** | "AI能代替人工吗？" | 三段式小标题都是问句，把"我想说的"变成"你想知道的" |
| **让数据说话** | "前端员工缩减七倍" | 精确数字代替主观判断 |
| **角色分类法** | "对于科技类公司……对于普通职能岗……对于应届生……" | 不同角色不同建议，读者自动对号入座 |
| **台阶式观点** | "最实用的是用AI基于个人事实完善简历" | 不说"你应该用AI"，说"最实用的是……" |

### 特别值得学的：角色分类的时机

文章在3个关键位置做了角色分类：
1. **开头**：科技类公司（AI精专人员）vs 普通职能岗（AI使用能力）
2. **中段**：广告设计（新标准是"销售能力≈专业能力"）vs 程序员（"vibe coding"）
3. **结尾**：AI+硬件交叉岗位（增量爆发）vs 初级岗位（大幅缩减）

**规律**：读者是被动阅读的，分类越早，读者进入自己角色的时间越早，停留越久。

## 四、语言痕迹分析

### 优点

1. **采访原话对比强烈** — "不会养虾的直接淘汰"这种HR原话冲击力极强
2. **用词精准** — "AI脑崩"、"Vibe Coding"、"有效补位"都是可转述的新词
3. **短句穿插** — "AI全民化→焦虑"、"方便→卷"等对比句式

### 问题

1. 段落过于密集，读起来有"采访记录拼贴"的感觉
2. 缺少个人视角的"我"字行文，作者隐身度太高，像一篇调查报告而不是文章

## 六、可复用技巧清单

1. ✅ **三问式结构**：用三个问号做小标题，把你想说的变成读者想知道的
2. ✅ **多源采访拼贴**：覆盖招聘方+求职方，让读者自动对号
3. ✅ **对比嵌套递进**：从效率→薪资→要求，逐步靠近读者利益
4. ✅ **HR原话做钩子**："不会养虾直接淘汰"——采访中的原话冲击力远超作者转述
5. ✅ **三个位置的角色分类**：开头、中段、结尾各做一次分类，持续锚定读者
