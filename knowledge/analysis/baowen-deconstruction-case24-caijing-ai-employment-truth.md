## 一、文章结构

### 五段递进式框架

```
引子：杨千的故事——互联网公司裁掉三分之二研发人员
↓
段落1：再就业3个月又没了，"一岗多能"渐成标配
  → 戴奈（运营）被裁→自由职业，设计师被AI替代
  → 阿亮（AI创业者）"6人公司产出过去50人的成果"
```

## 二、素材组织手法

### 手法1：**个人故事→数据→专家→政策**四层金字塔

每个段落的结构都是：
1. **一个活生生的人的故事**（杨千被裁、戴奈转行、陆远舟创业）
2. **一组精确数据**（月薪对比、裁员比例、岗位缩减数字）
3. **权威专家分析**（卓贤、皮萨里德斯）
4. **制度/政策层面的建议**（机器人税、社保改革）

**可学**：深度文最怕"空洞说理"，最怕"只有案例没有分析"。这篇的四层结构让读者**先"感受到"→再"相信"→然后"理解"→最后"行动"**。

### 手法2：**金句密度极高**

| 位置 | 金句 | 作用 |
|------|------|------|
| 开头 | "未来不在过去的延长线上" | 定调，格言感 |
| 中段 | "人离职，Skill继续" | 令人不寒而栗的画面 |
| 中段 | "过去，你靠一点天赋，能拿到一张实习门票……但现在这扇门被死死锁上了" | 情绪高潮 |
| 结尾 | "技术从不提前通知谁被淘汰，它只悄然重写规则" | 收束全文，余音 |

**规律**：每800-1000字安排一个金句，分别承担"定调→推进→高潮→收束"的功能。

### 手法3：**对比嵌套（三层）**

```
第一层：省钱对比
  老员工月薪3万 vs 三位应届生+AI月薪2.4万
第二层：效率对比
  需求排期两周→三天出原型
第三层：替代对比
  八年经验的技术"牛人"→会用Cursor的应届生
```

不只是比一次，而是**效率→成本→替代度**三层递进，每一层都在"加码"。

## 三、观点表达手法

| 手法 | 原文 | 效果 |
|------|------|------|
| **让采访对象说话** | "AI先消灭了年轻人'通过初级岗位成长为专家'的路径"（卓贤说） | 比作者说有力10倍 |
| **反直觉论断** | "体力劳动与蓝领岗位更能抵御AI冲击" | 打破白领优越感 |
| **概念命名** | "一岗多能""同事.skill" | 标签化方便记忆 |
| **提前消疑** | "但这并非板上钉钉"（高盛的补充） | 增加可信度，不自吹 |
| **带情绪的收束** | "如果你没有很多内涵，你只会被AI牵着走" | 让读者感受到紧迫 |

### 最值得学的：**"隐身作者"+"多声部"写法**

作者几乎不出现在文中，意见全部通过**采访对象的话**和**数据**来传递：

```
"如果你空有一身才华，却连上牌桌的机会都没了" → 陆远舟说
"一个月产出过去50人的成果" → 阿亮说
"6人公司产出赶上50人" → 数据说
"考虑征收机器人税" → 专家说
```

作者只在开头和结尾各写了一句自己的话（"未来不在过去的延长线上"、"技术从不提前通知"），其余全部隐身。

## 四、语言痕迹分析

### 优点

1. **开场画面感极强**："工位空了大片，从前吵吵闹闹的办公室，现在安静得可怕"——视觉+听觉双重冲击
2. **数据穿插精确**："月薪3万 vs 月薪2.4万"、"AI成本每月五六千"——不是笼统说"便宜"，而是**精确到千元级别**
3. **用"发现型"句式**："他很快发现""切身的观察在数据上也有体现"——代替"我们可以看出"
4. **情绪节奏控制**：每个故事讲完→停一停→加一句金句→继续推进

### 可改进

- 篇幅稍长（全文近8000字），容易阅读疲劳
- 三个采访对象的切换稍显突兀，缺少过渡段落
- 开头的"杨千"故事很好，但后续没有再出现杨千的视角，读者会期待他的结局
