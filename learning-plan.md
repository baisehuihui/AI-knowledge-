# 公众号文章学习计划（7天·500篇）

**启动时间**：2026-05-04 19:10
**结束时间**：2026-05-10 19:10

---

## 总体目标

- 泛读500篇公众号文章，建立"人味写作"的体感
- 精拆50篇，按5个维度深入分析
- 产出《人味写作核心笔记》，形成可复用的写作准则
- 对照学习成果，重写最新版文章

---

## 每日安排

### 第1天（5/4）今晚7点-睡前
**任务量**：泛读30篇 + 精拆3篇
**重点**：建立搜索渠道、找第一批好文章入账
- 确定要关注的公众号/信息源
- 通过微信搜索、推荐阅读等方式获取文章
- 泛读30篇，标记3篇精拆
- 产出当日学习笔记

### 第2天（5/5）全天
**任务量**：泛读80篇 + 精拆8篇
**重点**：重点关注开头手法、积累不同类型开头样本
- 专注拆解不同人的开头写法
- 记录每一类开头的优缺点

### 第3天（5/6）全天
**任务量**：泛读80篇 + 精拆8篇
**重点**：素材组织方式、段落衔接
- 关注真人作者如何把多个素材串起来
- 拆解"我想到的"式过渡 vs "此外/另一方面"式过渡

### 第4天（5/7）全天
**任务量**：泛读80篇 + 精拆8篇
**重点**：观点表达方式（如何避免说教）
- 关注"讲观点"和"说教"的分界线
- 拆解作者如何让读者自己得出观点

### 第5天（5/8）全天
**任务量**：泛读80篇 + 精拆8篇
**重点**：语言痕迹检测、AI句式替代
- 系统记录AI常用固定搭配和真人替代写法
- 建立"AI禁词/禁句式"清单

### 第6天（5/9）全天
**任务量**：泛读80篇 + 精拆8篇
**重点**：结尾方式、整体节奏
- 拆解不同类型结尾（留问题/小故事/反转/共情）
- 关注段落长短变化、节奏控制

### 第7天（5/10）全天
**任务量**：泛读70篇 + 精拆7篇（凑满500+50）
**重点**：整理输出+实践
- 整理《人味写作核心笔记》
- 用学习成果重写最新版文章
- 把学习笔记全部整理存知识库

## 每篇文章的拆解模板

每篇精拆文章，保存到知识库 `knowledge/articles/` 目录下，结构如下：

```
# 文章信息
- 标题：
- 作者/公众号：
- 链接：
- 阅读量/在看（如有）：

## 1. 开头分析
- 第一句话是什么？
- 用了什么手法？（具体动作/场景画面/矛盾冲突/设问/情绪带入）
- 效果如何？

## 2. 素材组织
- 用了几个素材/故事？
- 素材之间怎么连接的？
- 有没有"我"在其中串联？

## 3. 观点表达
- 观点是怎么给出的？（直接说/讲故事让读者自己悟/反问引导）
- 有没有说教感？为什么有/没有？

## 4. 语言痕迹
- 有AI常用句式吗？
- 有哪些特别"人味"的表达？
- 长短句、标点使用特点

## 5. 结尾分析
- 怎么收的？
- 留给读者什么感觉？

## 6. 整体评价
- 这篇最值得学习的地方
- 如果我来写，借鉴什么
```
