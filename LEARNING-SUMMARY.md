## 一、五大核心发现

### 🏗️ 1. 三层结构是骨架

好文章不是"第一第二第三"平铺，而是有分层递进的骨架结构：

- **第一层：现象/故事**（抓人往下读）
- **第二层：分析/拆解**（提供认知增量）
- **第三层：观点/行动指引**（让读者带走点东西）

**反面教材**：三段式平铺（第一、第二、第三）+ 每段字数对等 + 结尾总结

### 🧑 2. 不完美 = 人味

AI追求完美对齐——人不是。以下"缺陷"才是人的痕迹：

- 段落忽长忽短（一句成段 vs 长段叙事）
- 自嘲和跑题（"说回正题……"）
- 字数不对等（有的段落一行，有的换行多段）
- 突然的情绪爆发（"妈的"、"离谱"）

> **原则**：不完美不是错误，是人存在的证明

### 🔢 3. 数字顶一万句

- "100万字" → 比"大量写作"强100倍
- "299元保证金" → 比"收入不高"强100倍
- "1万阅读32块" → 比"收益不高"强100倍
- "3分钟生成" → 比"很快"强100倍

**铁律**：能用具体数字替换的形容词，一律替换。放数据出去，不带结论回来，让读者自己算账。

### 👤 4. "我"字扛全文

- 有"我"才有立场
- 有立场才有态度
- 有态度才有人味
- 没有人味的文章没有关注价值

**写法**：不是"这件事值得关注"，而是"我为什么关注这件事"。用"我"字做文章的承转逻辑线。

### 🔚 5. 不总结的结尾

最好的结尾不是总结，而是：

- **反问**："你呢？"
- **挑衅**："这很反常识——但就是事实"
- **留白**：戛然而止，让读者自己想
- **反讽**：用调侃替代说教
- **沉默**：一句收完，不补任何话

> **铁律**：每删一次"综上所述"，人味+1

## 二、AI味的本质与去AI味方法论

### AI味的本质不是词，是模式

| AI味特征 | 人味替代方案 |
|:---------|:------------|
| 每段整齐三段式 | 段落长短不一，一句成段 |
| 每个观点都平衡中立 | 大胆表态，有立场 |
| 结尾统一总结 | 反问/留白/反讽收尾 |
| 开头"随着……" | 直接场景切入/故事钩子 |
| 每段先结论后解释 | 先故事后结论/只故事不结论 |
| 没有"我"的叙述 | "我"字扛全文 |
| 每段字数对等 | 字数不对等，呼吸感 |
| 列表式1.2.3. | 自然的叙述流 |

### 去AI味实操检查清单（8项）

1. ✅ 去套话——删掉"随着""值得一提的是""需要指出的是"
2. ✅ 打破结构——每段整齐三段式 → 长短不一
3. ✅ 改列表——1.2.3. → 自然叙述
4. ✅ 换结尾——"综上所述" → 反问/留白
5. ✅ 加"我"——整篇文章有"我"做逻辑主线
6. ✅ 大胆表态——不怕得罪人，有立场
7. ✅ 读出声——读一遍，能听出AI味的就改
8. ✅ 删最后一段——大部分文章的结尾删掉更好

### 正确的AI使用姿势

- AI做：找选题 + 起标题 + 改错字
- 自己做：故事 + 观点 + 情绪
- 顺序不能反：先人工跑通小闭环，再用AI放大

- **好内容的标准在回归**：选题占50%、标题占20%、开头占10%、正文占20%

- 70%的"年赚200万"来自299元保证金，本质是微商裂变
- 真正跑通的OPC共性：先人工跑通小闭环，再用AI放大

- 投资分析师被降薪30%后裁掉
- 通信外企整个部门被端掉
- 翻译从业者收入腰斩
- 设计师行业AI替代率已达30%-50%
- 但"被替代"不是一夜之间——是温水煮青蛙

### 好内容公式

> **热点事件 × 独特框架 × 真实感受 = 爆款**

缺任何一个因子都成不了爆款：
- 有热点没框架 = 跟风搬运
- 有框架没感受 = 教科书
- 有感受没热点 = 没人看
- 三个都有 = 爆款

## 六、写作铁律

1. **每条关键信息必须有可查出处**——人名、时间、数据、来源一样不能少
2. **禁止模糊用词**——"大概""不少""很多""据说" → 具体数字
3. **禁止编造**——没有核实的信息不写，宁可少写也不糊弄
4. **每个关键数据标注来源**——"据XX报道"
5. **数字顶一万句**——放数据出去，不带结论回来
6. **不总结的结尾**——反问/留白/反讽 > "综上所述"
7. **人味大于完美**——不完美=人存在的证明
8. **"我"字扛全文**——第一人称立场，不做转述者
